PGE Miner

Plataforma web para busca inteligente de documentos jurídicos da PGE-SC. Combina autenticação corporativa, integrações SGPE/EPROC, Elasticsearch para pesquisa full-text e UI responsiva para revisar pareceres, atos e dispensas de recurso.

Destaques

Pesquisa avançada

Filtros por datas, assinaturas, núcleos, tipos de citação/intimação e status de revisão, com inteiro teor em PDF.

Pareceres e Word

Consulta e edição de pareceres, leitura inline e conversão rápida para Word com histórico de ações.

Fluxos PROCON/SAF

Gestão de dispensas de recurso com revisão, marcação e exclusão em cartões, focado em velocidade de triagem.

Revisão lado a lado

Formulário e PDF simultâneos para atualizar assunto, núcleo e ementa sem perder o contexto do documento.

Stack e Arquitetura

Backend

Python 3, Django 3.2, DRF, SimpleJWT, CORS; Gunicorn em produção.

Django DRF SimpleJWT Gunicorn

Frontend

Next.js 13 (React 18, TypeScript) com Chakra UI, React Table, React Hook Form, CKEditor 5, React-PDF-Viewer.

Next.js 13 React 18 TypeScript Chakra UI CKEditor 5

Pesquisa e Dados

Elasticsearch 7.17 (django-elasticsearch-dsl/dsl-drf), PostgreSQL e Oracle via cx-Oracle.

Elasticsearch PostgreSQL Oracle cx-Oracle

Integrações

SGPE (login/processo/tramitação/documentos), EPROC 1G/2G via WSDL, LDAP corporativo e e-mail transacional.

SGPE EPROC 1G/2G LDAP SMTP

Docs/OCR

PDFMiner, PyPDF2, pdf2image, pytesseract, pdf2docx e reportlab para extração e conversões.

pytesseract pdf2image pdfminer.six

Infra

Docker/Compose (dev/prod), volumes para estáticos/media, server Node/Next custom.

Docker Compose Node server
Galeria
Como rodar (dev)
Backend
docker-compose up (em backend-miner) e depois criar superusuário + indexar: python3 manage.py search_index --rebuild
Frontend
yarn install && yarn dev (ou compose na raiz para subir tudo).
Variáveis
Configure SGPE, EPROC, LDAP, bancos e SMTP via .env (não versionar dados sensíveis).
