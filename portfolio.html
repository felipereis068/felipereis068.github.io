Projetos: Miner e Plataforma

Dois produtos distintos: Miner (busca inteligente de documentos jurídicos) e Plataforma PGE (hub de ferramentas: pautas, atos normativos, reservas, integrações SOAP). Clique nas imagens para ampliar.

Miner

Busca inteligente de documentos da PGE-SC com Elasticsearch, integrações SGPE/EPROC e UI em Next.js.

Destaques

Pesquisa avançada

Filtros por datas, assinaturas, núcleos, citações/intimações; inteiro teor em PDF.

Pareceres e Word

Consulta/edição, leitura inline e conversão rápida para Word.

Fluxos PROCON/SAF

Gestão de dispensas com revisão, marcação e exclusão em cartões.

Revisão lado a lado

Form + PDF simultâneos para editar assunto, núcleo e ementa.

Stack e Arquitetura

Backend

Python 3, Django 3.2, DRF, SimpleJWT, CORS; Gunicorn em produção.

DjangoDRFSimpleJWTGunicorn

Frontend

Next.js 13 (React 18, TS) com Chakra UI, React Table, React Hook Form, CKEditor 5, React-PDF-Viewer.

Next.js 13React 18TypeScriptChakra UI

Pesquisa e Dados

Elasticsearch 7.17, PostgreSQL, Oracle via cx-Oracle; buscas abastecidas por scripts agendados.

ElasticsearchPostgreSQLOracle

Integrações

SGPE (login/processo/tramitação/documentos), EPROC 1G/2G via WSDL, LDAP, SMTP; uso de SOAP (Zeep) para chamadas externas.

SGPEEPROCSOAP/ZeepLDAPSMTP

Docs/OCR

PDFMiner, PyPDF2, pdf2image, pytesseract, pdf2docx, reportlab.

pytesseractpdf2imagepdfminer.six

Infra

Docker/Compose (dev/prod), Kubernetes (Miner) gerenciado via Rancher, Harbor para versionar imagens, Git, Node server custom e Nginx como proxy/reverso.

DockerComposeKubernetesRancherHarborGitNginx

Plataforma PGE

Hub de ferramentas: pautas de audiência, atos normativos, reservas de salas e integrações SOAP parametrizadas.

Destaques

Hub unificado

Dashboard com busca, categorias, favoritos e notificações.

Pautas de audiência

Filtros por órgão, relator, assunto, tipo de sessão e painel consolidado usando a API para integração dos dados.

Atos normativos

Consulta e gestão com UI responsiva Radix/shadcn.

Reserva de salas

Status em tempo real (Firebase) e verificação de conflitos.

Segurança e SSO

JWT httpOnly, opção de LDAP corporativo, CORS/CSRF hardening.

Integrações SOAP

Zeep/xmltodict com WSDLs, serviços/tribunais, manipulação de envelopes.

Stack e Arquitetura

Backend

Django + DRF, SimpleJWT, WhiteNoise, CORS, logging configurável.

DjangoDRFSimpleJWTWhiteNoise

Frontend

Next.js + React 18 (TS), Radix UI/shadcn, lucide-react, react-hook-form, recharts.

Next.jsReact 18TypeScriptRadix UI

Dados & Integrações

PostgreSQL 16; Zeep/XML (SOAP) com WSDLs/headers/payloads dinâmicos; Firebase (reservas).

PostgreSQLZeep/SOAPxmltodictFirebase

Infra

Docker Compose (dev/prod) separado por serviço; Rancher para gerenciamento de contêineres; Harbor para versionamento de imagens; Git; Nginx na borda; .env por ambiente.
